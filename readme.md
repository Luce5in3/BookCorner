## book_copies 表 status 字段对应状态

| 数值 | 状态     | 说明                 |
| ---- | -------- | -------------------- |
| 0    | 注销     | 副本已从系统中注销   |
| 1    | 在馆     | 可借，在图书馆内     |
| 2    | 借出     | 已被读者借走         |
| 3    | 预约锁定 | 被预约，等待读者取书 |
| 4    | 丢失     | 副本丢失             |

## 品相 (condition) 字段

| 数值 | 品相 |
| ---- | ---- |
| 1    | 全新 |
| 2    | 良好 |
| 3    | 一般 |
| 4    | 破损 |
